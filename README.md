# BigLaw - 開放法律判決智慧檢索與分析應用

因為[司法院法學資料檢索系統](http://jirs.judicial.gov.tw/Index.htm)並不是很好用

因此我們想辦法將資料爬下來並且整理  
自己製作圖表與使用搜尋引擎提供檢索
